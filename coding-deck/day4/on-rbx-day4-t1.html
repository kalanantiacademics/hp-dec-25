Misi: Membuat Sistem Teleportasi Antar Pulau dan Layar Ending!

Misi Sebelumnya: Recap

Kemarin kita sudah berhasil membuat dua hal penting:


  • ‚úÖ Leaderstats: Papan skor untuk mencatat jumlah Artefak.
  • ‚úÖ Collecting System: Mengambil barang dengan tombol "E".

Hari ini kita akan menggunakan skor tersebut untuk membuka pintu ke level selanjutnya!

Misi 3: Membuat Pintu Ajaib (Teleportasi)

Sekarang kita akan membuat "Kendaraan" untuk pindah pulau.

Tanpa teleportasi ini, pemain akan terjebak di pulau pertama selamanya.


üéØ Misi: Membuat sistem teleportasi agar pemain bisa pindah dari Jawa -> Kalimantan -> Sumatera.

Syarat: Pemain hanya boleh pindah kalau SEMUA barang di pulai itu sudah diambil!

Langkah 1: Melanjutkan Kode 'Wasit'

Kita tidak perlu membuat script baru. Kita akan melanjutkan coding di script Main yang sudah kita buat.


  1. Buka script bernama Main di dalam folder ServerScriptService.
  2. Pergi ke baris paling bawah kode yang sudah ada.

Kita akan menambahkan kode baru khusus untuk teleportasi di sana.

üìù

Langkah 2: Logika Pintu Terkunci

Bayangkan teleportasi ini seperti Pintu Pemeriksaan di Bandara.

Pintunya berkata: "Eits, kamu belum boleh lewat! Kantongmu masih kosong. Ambil dulu semua barang di pulau ini!"


Cara Kerja Kode:

  • Cek Folder Artefak: Script akan mengintip ke dalam folder Artefacts -> Jawa.
  • Hitung Barang: Apakah di dalam folder masih ada isinya?
  • Jika Masih Ada: Pintu teleportasi sembunyi.
  • Jika Kosong (0): Pintu teleportasi muncul dan pemain bisa lewat!
Langkah 3: Mengisi Kode Tujuan

Kita harus mengatur rute perjalanannya (seperti tiket pesawat).


Cari bagian kode yang mengatur tujuan (Destination) dan isi pasangannya:

  • Jika di Jawa, tujuannya ke -> Kalimantan.
  • Jika di Kalimantan, tujuannya ke -> Sumatera.

Hati-hati! Penulisan nama pulau harus sama persis dengan nama folder.

if daerah == "Jawa" then tujuannya = "Kalimantan"

Langkah 4: Mengatur Tombol Interaksi

Sama seperti saat mengambil barang, kita akan membuat tombol "E" muncul di pintu teleportasi.


Di dalam kode, kita atur tulisan tombolnya:

Action Text: "Teleport"

Object Text: "Teleport ke [Nama Tujuan]"


Jadi nanti saat mendekat, akan muncul tulisan: "E - Teleport ke Kalimantan".

Langkah 5: Uji Coba (Testing)

Ini bagian paling seru! Ayo kita coba mainkan.


  1. Klik Play.
  2. Jangan ambil barang dulu. Coba lari ke tempat teleportasi.
  3. Hasil: Seharusnya TIDAK ADA tombol yang muncul.
  4. Sekarang, ambil semua barang di pulau Jawa.
  5. Lari kembali ke tempat teleportasi.
  6. Hasil: Muncul tulisan "Teleport ke Kalimantan".
  7. Tekan E. Wush! Kamu sampai di Kalimantan!

Misi 4: Layar Tamat (Ending GUI) üèÜ

üéØ Misi: Memunculkan pesan "Selamat, Misi Selesai!" secara otomatis di layar pemain ketika mereka berhasil sampai di pulau terakhir (Sumatera) dan barangnya habis.


Bayangkan ini sebagai Piala di garis finish. Tanpa ini, pemain tidak tahu kalau gamenya sudah tamat!

Langkah 1: Modifikasi Script 'Main'

Kita masih bekerja di script "Main" yang sama.


  1. Buka script Main.
  2. Cari kode teleportasi terakhir (else if daerah == "Kalimantan").
  3. Hapus 3 huruf end paling bawah.

Kita hapus 'end' karena kita mau menambahkan kondisi baru di bawahnya: "Jika di Sumatera, maka..."

Langkah 2: Membuat GUI dengan Kode

Biasanya kita membuat tampilan (GUI) secara manual, tapi kali ini kita akan menggunakan mantra (kode) agar otomatis!

Kita menyuruh script melakukan ini:

  • Buat Layar (ScreenGui): Bernama 'Pulau Selesai UI'.
  • Buat Kotak (Frame): Di tengah layar dengan warna pilihanmu (Klik palet warna di script!).
  • Tulis Pesan (TextLabel): "SELAMAT! KAMU MENYELESAIKAN MISI!" (Ganti Font jadi Bangers biar keren).
