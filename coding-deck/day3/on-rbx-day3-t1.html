Membuat Papan Skor (Leaderstats)

Pertama, mari kita bahas tentang Leaderstats:


üéØ Tujuan Misi: Kita akan membuat sistem "Papan Skor" agar pemain bisa melihat jumlah artefak yang berhasil dikumpulkan dari daerah Jawa, Kalimantan, dan Sumatera.

Langkah 1: Membuat Tempat Kode (Script)

Pertama, kita harus menyiapkan tempat untuk menulis mantra (kode) kita.


  1. Lihat bagian Explorer di sebelah kanan layar.
  2. Cari folder bernama ServerScriptService.
  3. Klik tanda tambah (+) di sebelahnya, lalu pilih Script.

Ganti Nama Script

Penting! Ubah nama script tersebut menjadi Leaderstats.


Tips: Gunakan huruf 'L' Besar (Kapital) di awal.


Kenapa? Agar kita mudah menemukannya nanti saat dibutuhkan.

Langkah 2: Mulai Menulis Kode

Sekarang kita akan memasukkan perintah agar komputer mengerti keinginan kita.


Hapus tulisan print("Hello world") yang muncul otomatis.

Lalu, tulis kode (atau salin dari starter code jika disediakan).

Apa isi kodenya? (Penjelasan Konsep)

Bayangkan kita sedang membuat tas ransel untuk setiap pemain yang masuk ke dalam game.


Kita membuat fungsi (function) untuk membuat "tas" bernama leaderstats.

Tas ini wajib memiliki nama khusus agar Roblox mengenulinya sebagai papan skor.

üéí
Langkah 3: Menambahkan Kategori Skor

Di dalam "tas" tadi, kita perlu menyiapkan 3 kantong kecil untuk menyimpan nilai skor masing-masing daerah.


Kita membuat 3 IntValue (Nilai Angka) baru di dalam kode kita.

Beri Nama Nilai!

Beri nama ketiga nilai tersebut sesuai nama daerah artefak kita:

  • Jawa
  • Kalimantan
  • Sumatera

‚ö†Ô∏è PERINGATAN PENTING!

Penulisan nama harus SAMA PERSIS dengan nama folder artefak yang sudah kamu buat sebelumnya.


Perhatikan huruf besar dan kecilnya (Case Sensitive):

  • Tulis Jawa (Bukan jawa)
  • Tulis Kalimantan (Bukan kalimantan)
  • Tulis Sumatera (Bukan sumatera)


Kenapa? Karena komputer sangat teliti. Beda satu huruf saja, kodenya tidak akan bekerja!

Langkah 4: Uji Coba (Testing)

Ayo kita cek apakah papan skornya sudah berhasil dibuat!


  1. Klik tombol Play di Roblox Studio.
  2. Lihat di pojok kanan atas layar (daftar pemain).
  3. Apakah di samping nama kamu sudah muncul tulisan Jawa, Kalimantan, dan Sumatera?

Pastikan angkanya semua masih 0 (Nol) karena belum ada yang dikumpulkan.

üéâ Selamat!

Kamu sudah berhasil membuat papan skor.


Saat ini skornya masih 0 karena kita belum membuat kode untuk mengambil barangnya.

Kita akan melanjutkannya di langkah berikutnya!

Misi 2: Membuat "Wasit" (Main Script)

Jika di modul sebelumnya kita sudah membuat "Papan Skor" (Leaderstats), sekarang kita akan membuat aturan mainnya.


Apa bedanya?

Misi 1 (Leaderstats): Kita menyiapkan wadahnya (Keranjang Skor).

Misi 2 (Main Script): Kita membuat "Wasit" yang bertugas: "Jika pemain mengambil Blangkon, maka masukkan angka 1 ke keranjang Jawa."

Modul Roblox Tema 1:
Sistem Pengambilan Artefak

üéØ Misi: Membuat artefak di dalam game bisa diambil dengan menekan tombol "E", dan skor di papan skor otomatis bertambah.

Langkah 1: Membuat Script Utama

Kita membutuhkan satu script sakti yang bisa mengontrol semua barang sekaligus.

Dengan script ini, kita tidak perlu memasukkan kode satu per satu ke dalam setiap barang.


  1. Pergi ke ServerScriptService.
  2. Klik tanda tambah (+) dan pilih Script.
  3. Ganti nama script tersebut menjadi Main.

"Main" artinya Utama. Ini adalah script pengendali permainan kita.

üëë

Langkah 2: Menulis Kode (Logika Perulangan)

Di dalam script ini, kita akan menggunakan teknik "Looping" (Perulangan).


Bayangkan script ini seperti robot yang rajin mengecek semua barang satu per satu, lalu menempelkan tombol "Ambil" di atasnya secara otomatis.

Bagian Penting dalam Kode:

1. Mencari Folder Artefak

Kita memerintahkan script untuk melihat ke dalam folder "Artefacts" tempat kita menyimpan semua model.

Bahasa mudahnya: "Robot, tolong cek semua barang yang ada di dalam folder Artefacts ya!"


2. Membuat Tombol Interaksi (ProximityPrompt)

Kita akan memasang tombol canggih bernama ProximityPrompt.

Nantinya, saat pemain mendekat ke artefak, akan muncul tulisan "Collect" dan tombol interaksi "E".

3. Menghubungkan ke Leaderstats (Poin PENTING!)

Di sinilah script ini berkomunikasi dengan Leaderstats.


Saat tombol ditekan, script akan menjalankan perintah:

"Cari Leaderstats milik pemain ini. Lihat barang apa yang diambil. Jika yang diambil dari daerah Jawa, maka tambahkan skor di papan Jawa sebanyak 1 poin."


player.leaderstats[NamaDaerah].Value = player.leaderstats[NamaDaerah].Value + 1

Langkah 3: Mengatur Nama Barang (Penting!)

Agar script "Wasit" ini tidak bingung, nama barang harus jelas.


Pastikan nama folder Artefak (Jawa, Kalimantan, Sumatera) tulisannya SAMA PERSIS dengan yang ada di script Leaderstats.


Ingat, huruf besar dan kecil sangat berpengaruh! (Jawa ‚â† jawa).

Langkah 4: Uji Coba (Testing Time!)

Ayo kita buktikan apakah sistemnya bekerja!


  1. Klik Play.
  2. Dekati salah satu barang (misalnya Blangkon). Apakah muncul tulisan "E - Collect"?
  3. Tekan tombol E di keyboard.
  4. Cek Papan Skor (Leaderstats) di pojok kanan atas. Apakah angka di sebelah "Jawa" berubah dari 0 menjadi 1?

Jika ya, selamat! Kamu sukses!
